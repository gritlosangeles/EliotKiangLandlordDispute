# Professional Timeline Site

Static website for Cloudflare Pages.

## Cloudflare Pages settings

- Framework preset: None
- Build command: leave blank
- Build output directory: /

## Updating the site

Edit `index.html` to add real timeline entries.  
Edit `styles.css` to adjust the visual design.

- `index.html`
- `styles.css`
